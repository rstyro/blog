# kali  安装中文拼音

> 听别人说，kali 很牛掰牛掰，新手不知道牛在哪。装个拼音都不会，尴尬啊。

### 1、安装过程如下：
+ 1、敲命令：apt-get install ibus ibus-pinyin
+ 2、在设置的-->区域语言-->添加汉字拼音
+ 3、重启kali 系统，可以用拼音了。
